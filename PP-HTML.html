  2. "head":

    3. TAG é utilizada para especificar os dados META referente ao programa, considerada a cabeça do sistema html, onde ela inclui o nome da página, suas dependências como JavaScript e CSS por exemplo, uso da fonte, etc.

  5. "base":

    6. É utilizada para especificar a URL base do seu site, onde é responsável pela linkagem com os links internos.

  6. "meta":

    7. Essa tag é utilizada para conformar o programa/página a determinadas configurações, como de linguagem, como a escala de visualização em diferentes dispositivos, etc. Também é útil para mencionar o autor da página, palavras-chaves, datas de publicação, etc. Exemplificado no tópico 02.


  7. "link":

    8. Essa tag é responsável para implementar um link externo no programa/página, do qual é possível acessar pela página programada. Também utilizada para incluir os stylesheets.


    O "target="_blank" é um atributo para que ao clicar no link, se abra numa nova aba.

  8. "style":

    9. A tag style pode ser utilizada como uma alternativa de uma stylesheet externo, como CSS, ou com a finalidade de complementa-la. Inclui as informações de aparência da página.

    10. É utilizada para adicionar códigos snippet, tipicamente em JavaScript, para tornar a página dinâmica. Também pode ser usada para linkar um script externo.

    Guia de Estrutura HTML

  10. "h1...h6":

    11. Seis variações de escrita de cabeçalho. Onde o "h1" possui o maior tamanho de fonte e o "h6" o menor.

  11. "div":

    12. O conteúdo de uma página é geralmente dividido em blocos, especificados pela tag div.

  12. "span":

    13. Esta tag insere elementos de linha, como uma imagem. ícone e emotion, sem estragar a formatação/estilo da página. Ou seja, implementa em uma condição de bloco em linha, com margem, padding mínimos. Muito utilizado no envelopamento do conteúdo desejado e aplicar parâmetros desejados apenas no conteúdo envelopado.

  13. "p":

    14. Texto sem formatação é colocado dentro dessa tag. (Exemplo: Parágrafo, utilizado no próprio conteúdo deste projeto).

  14. "br":

    15. Quebra de linha para uma página (Próxima linha). Utilizando quando você quer escrever em umma nova linha.

  15. "hr":

    16. Parecida com a tag "p", mas em adição além a pular linha essa TAG adiciona uma linha barra horizontal indicando o fim da seção.

  46. "picture":

    47. É uma tag importante, é responsável por deixar a imagem dinâmica dentro do site, em diferentes dispositivos mobile, desktop ou TV, referente ao tamanho, deixando-as sempre em imagem cheia independente do tamanho da tela do dispositivo. Utilizando propriedades da tag picture.

    1. A ordem dos comandos interfere no resultado final. Portanto, a imagem maior de ser implementada normalmente com a tag "<img src="" alt="" >".

    2. Logo na linha de cima utilizaremos o comando source, usando o atalho escolheremos a tag "<source: media: type >"

    3. Utilizando a tag source:media:type acima da linha atribuida com a maior imagem (img), atribuimos "source media="(max-width:tamanho da maior foto em px+50px" srcset="url da imagem média.png" type="image/tipo.da.imagem que no caso é png"

    4. Repetindo o passo anterior, mas agora para a imagem média para a pequena. Ainda utilizando a tag source:media:type acima da linha atribuida com a imagem média (source), atribuimos "source media="(max-width:tamanho da foto média em px+50px" srcset="url da imagem-pequena.png" type="image/tipo.da.imagem que no caso é png"

  47. Para áudios, tag "audio":

    48. Para incorporar um áudio, há dois métodos um simples, e outros mais elaborado, pela razãoo de que há navegadores que suportam determinados formatos e outros não suportam, então vamos analisar esses dois métodos:

    1. Basta usar a tag audio. Salientando a importância das propriedades atribuidas como src="" para localizar o áudio na pasta e controls que irá abrir os controles (play/pause|volume|velocidade) na página html, havendo também outras, como, loop para reiniciar automaticamente, mas vai do seu desejo, logo:
      <audio src="local-na-pasta" controls autoplay></audio>
    2. Na forma mais elaborada temos a ciência dos formatos de áudios, se é mpeg, ogg, wav,... então para não correr o risco de um determinado navegador não executar o áudio usamos:
      <audio preload="metadata" (podendo ser "auto" ou "none") controls loop>
      <source src="localização-do-audio" type="audio/formatodoaudio(mpeg|ogg|wav">
      Auto quando abrir no navegador ele irá primeiramente carregar o arquivo todo | metadata irá carregar as informações primordialmente, só depois de dar o play que irá carregar a música em si | none não irá carregar nada inicialmente.
  48. Incorporação de vídeos, tag "video":

    49. Como nos áudios há também maneiras de incorporar vídeos no seu domínio, a primeira delas é hospedado localmente, onde o vídeo ficará hospedado no seu próprio servidor. A segunda opção, mais recomendado por conta de custos, é inserindo os vídeos de uma outra plataforma online, por exemplo "YouTube" ou "Vimeo".

    1. Como primeira opção que é hospedando o site do próprio servidor, temos, também duas opções de incorporar no HTML, como nos áudios referente aos formatos de vídeos, temos da maneira simples e mais elaborada.
      • Pela maneira mais simplificada podemos incorporar desta maneira:
        <video src="local-do-video" controls width="VALORpx"></video>
      • Segunda maneira, mais elaborada, reduz erros de um navegador não reproduzir um vídeo por causa de incompatibilidade de formatos, então:
        <video controls width="560px">
        <source src="midia/meu-video-mod.mp4" type="video/mp4">
        <source src="midia/meu-video-mod.ogg" type="video/ogg">
        <source src="midia/meu-video-mod.webm" type="video/webm"></video>
        Observação: Não quer dizer que irá abrir os três vídeos, o navegador indentificará qual tem a compatibilidade na ordem de cima para baixo e disponibilizará na tela para o usuário.
    2. E pela opção de incorporar vídeos de uma hospedagem externa, como exemplo Youtube. Basta ir na opção compartilhar e selecionar a opção "Incorporar" e copiar o link disponibilizado e colar no seu HTML, exemplo abaixo:
      <iframe width="560" height="315" src="https://www.youtube.com/embed/xovRYgwapkI" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>
