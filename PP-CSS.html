Guia de Referência CSS3

Estilo em cascatas, focado na imagem do site. Aqui iremos abordar maneiras, dicas e formas de trabalhar com CSS atrelado ao HTML. Bons estudos.

    Há estruturas de incorporação do CSS, Inline, Internos e Externos

    O estilo CSS inline é uma forma de implementar a aparência em cada linha específica, assim implementando apenas na linha em que estiver trabalhando.

    O estilo CSS interno, é uma maneira mais vantajosa comparada a implementação do CSS Inline, uma das vantagens é a organização, pois é uma opção em que pode separar a estrutura HTML da CSS no mesmo programa.

    Por fim, há maneira mais recomendada entre a citadas acima, que o CSS Externo, pois toda a estrutura CSS é construida em uma outra página, por fim é linkada do HTML para CSS, onde é atribuida as propriedades do CSS na estrutura HTML, deixando assim ambos mais organizados.

    Obeservação: É necessário atribuir alguns valores para não dar inconsistências no CSS, por exemplo, as acentuções brasileiras e simbolos, sim @charset "UTF-8"; é necessário.

    Guia de Cores:

  1. A tag background-color Cores de fundo:

    2. Se refere a modificar a cor de fundo do site. Sendo ele um estilo (style="") alicado em uma camada seja no body ou footer,

    <body style="background-color: blue;"> - Sendo que o blue é um valor da propriedade background-color.

  2. A tag (style="color: Valor;"):

    3. É a tag responsável em alterar a cor, muito usando para alterar cor de textos. Exemplo:
    <h2 style="color: darkred;">Capítulo 1.2</h2>

    Guia de Textos

  3. A tag style="text-align: VALOR;"

    4. Serve para fazer o alinhamento do texto na página web, mas há diversos alinhamentos. Como;

    • justify: Serve para deixar as linhas alinhadas (rente) nas duas extremidades da página.
    • center: Serve para deixar o texto centralizado
    • end: Posiciona-se no final do conteúdo.
    • left: Posiociona-se a esquerda da página web
    • right: Posiociona-se a direita da página web
    • start: Posiociona-se no início da página web
  4. A tag "font-size: VALORpx;":

    5. Serve para alterar o tamanho das letras.

  5. A tag "font-family: FONTE-DE-LETRA;":

    6. Serve para trocar a fonte das letras. Fonte essas instaladas na biblioteca do programa VSCode.

  6. A tag "style="text-decoration:POSIÇÂO-DA-LINHA;"

    7. Serve para inserir uma linha horizontal, afim de decorar um determinado conteúdo desejado, podendo ser um;

    • underline: Linha horizontal abaixo das palavras
    • line-through: Linha horizontal no meio das palavras
    • overline: Linha horizontal acima das palavras

    Guia de Seletores

    Há como seletores, podendo ser selecionado a partir de algumas tags, como, h1...h6, parágrafos, listas, div's, section's, body, main, footer, header, aside, article, nav's, links, e outros. Como também utilizando seus indicadores, personalizando seus respectivos seletores, como id e class, e seus pseudos; pseudoclass e pseudoelemento

  7. Personalizando apenas os conteúdos H1, por exemplo, h1{ ~personalização que deseja criar, lembrando, apenas nas tags h1~ }

    8. Configurando seletores, é simples de modo geral, conforme a sintaxe acima, basta seleciona-lo e abrir/fechar chaves e aplicando a personalização desejada entre as chaves.

  8. Utilizando identificação, id

    9. É possível diferenciar um determinado elemento h1 de um outro h1 e aplicar estilo diferente dos demais, utilizando um identificador. Observação no HTML é id / no CSS é #. Por exemplo;

    Sintaxe no HTML:
    <h1 id="principal">TEXTO EXEMPLICATIVO</h1>

    Sintaxe no CSS:
    #principal{ ~personalização de estilo que deseja~ }

  9. Utilizando as classes, class;

    10. Também com o intuito de diferenciar alguns elementos, podemos personalizar não apenas um, mas sim vários elementos, com isso utilizamos o identificador class. Observação no HTML é class / no CSS é . . Conforme exemplo abaixo;

    Sintaxe no HTML:
    <h3 class="guias">TEXTO EXEMPLICATIVO</h3>

    Sintaxe no CSS:
    .guias{ ~personalização de estilo que deseja~ }

    Utilizando os pseudos: pseudoclass e os pseudoelemento

  10. Utilizando os pseudo-classes;

    11. A pseudo-classe deve está relacionada a um estado de um determinado elemento ou a uma classe, no caso, se ele está ativo, se está marcado, se está habilitado, etc. Observação, no CSS é identificada com o :. A seguir veremos algumas pseudo-classes muito utilizadas.

    • hover: Para quando passar o cursor do mouse por cima, criar um estado personalizado.
    • visited: Para quando o site já ter sido visitado apresentar um estado personalizado.
    • active: É acionado no momento em que é "clicado", acionando assim um estado personalizado desejado.
  11. Utilizando o pseudo-elemento

    12. O pseudo-elemento é muito utilizado para personalizar as periferias do conteúdo. onde se utiliza o before/after. É possível também ser aplicado em classes, não apenas em elementos. Obervação, no CSS utiliza-se o :: A seguir alguns pseudo-elementos muito utilizadas;

    • before: Tudo antes do conteúdo selecionado.
    • after: Para tudo que vem depois do conteúdo selecionado.
  12. Utilizando o seletor filho:

    13. É também uma forma de selecionar um determinado elemento que esteja dentro de um seletor, desta forma a seleção também é específica, pois só abrange os elementos, de sua escolha, que estejam dentro do seletor, também de sua escolha.     Guia Modelo de Caixas

    Na finalidade de organização, se refere ao box-model em que possibilita uma formação de caixas dentro de outra caixa que por sua vez pode ser inserida em mais outra caixa e assim consecutivamente, esse modelo chama-se de "aninhamento". Também é tratado tudo referente ao box-sizing, do qual se tem como sua estrutura de dentro para fora, o conteúdo, padding, border, outline e margin, respectivamente.

    Anatomia de Caixa

    Tudo que é está em um site são conjuntos de caixas que estam inseridas os conteúdos, essas caixas tem suas medidas que podem ser personalizadas, vamos ver a anatomia, estruturas das caixas;

    • height: Altura da caixa.
    • width: Largura da caixa.
    • conteúdo: Como o próprio nome diz, é a informação que queremos apresentar.
    • padding: É o espaço entre o conteúdo e a borda (border), espaço interno.
    • border: É a borda, que é uma linha que circunda o conteúdo
    • outline: É o contorno, pelo lado externo da borda, caso seja necessário.
    • margin: É o espaço externo da caixa, ao redor da borda, para que a borda da caixa não tenha contato com outras bordas de outras caixas.

    Tipos de Caixa

    Box-level: Que é uma estrutura baseada em caixa que é gerada na linha abaixo de outra caixa e assim sucessivamente. Que tem como largura a largura da tela por inteiro.
    Por exemplo;

    • <div>
    • <h1-h6>
    • <p>
    • <main>
    • <header>
    • <nav>
    • <article>
    • <aside>
    • <footer>
    • <form>
    • <video>

    Inline-level: Que tem como estrutura uma caixa ao lado de outra caixa. Não há a quebra de linha. Que tem como largura apenas o que é necessário.
    Por exemplo;

    • <span>
    • <a>
    • <code>
    • <small>
    • <strong>
    • <em>
    • <sup>-<sub>
    • <label>
    • <button>
    • <input>
    • <select>
